# Vibe Coder Simulator campaign site

Public deployment repository for the Vibe Coder Simulator campaign page and playable prototype.

Visit the live site at [victortanws.github.io/vibe-coder-simulator-site](https://victortanws.github.io/vibe-coder-simulator-site/).

The deployable files in this directory are generated from the main local project with:

```sh
bash scripts/build-public-site.sh
```
